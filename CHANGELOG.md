# Changelog

All notable changes to the Gospel RPG iOS app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-03

### Added
- SNES-style Gospel RPG with pixel art assets
- OverworldView with Simon/Peter map
- Atmospheric effects and visual richness
- MusicEngine with SNES-style music
- Game persistence and state management
- App icon and launch screen
- GitHub Actions CI workflow for iOS builds
- Pre-push hook auto-install script
- LICENSE and security gitignore patterns

### Fixed
- Black screens during dialogue, floating character, and hold-to-move controls
- Timer/polling/state wiring issues
- onChange updated to iOS 17 two-parameter form
- Required init(from:) moved to class body for GameCharacter Decodable conformance
- Hardcoded path replaced with relative resolution

### Theology
- Jesus made undefeatable per Bellarmine ruling
- Invented Jesus dialogue converted to narrator text
- Non-scriptural dialogue corrected
- Bartholomew scene, holy water, and Markan Scripture fixes applied
